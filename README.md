# AdvancedTech North - Landing Page

Landing page desenvolvida com HTML e CSS puro, focada em layout moderno, animações e responsividade.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Pseudo-elements
- Transitions & Transforms
- Animations

## 🎯 Objetivo

Praticar construção de layout completo sem uso de JavaScript.
